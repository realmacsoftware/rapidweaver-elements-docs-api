# Syntax Highlighting

The **Elements VSCode Extension** brings powerful, purpose-built tooling to Visual Studio Code for developers working with **RapidWeaver Elements**. It enhances your editing experience with features tailored specifically to Elements projects and helps you stay focused, productive, and confident while building components and packs.

Whether you’re new to Elements or a seasoned pack creator, this extension adds real value to your development workflow.

### What the Extension Does

The RW Elements VSCode Extension provides essential developer features to make editing, navigating, and building Elements projects easier. Key capabilities include:

* Syntax highlighting and language support for Elements files
* Smart IntelliSense and suggestions
* Helpful snippets and shortcuts
* Integration with Elements tooling workflows

The extension blends seamlessly into Visual Studio Code, improving productivity without adding unnecessary complexity.

### Installing and Using the Extension

Installation is quick and easy:

1. Visit the extension’s page on the Visual Studio Marketplace: [https://marketplace.visualstudio.com/items?itemName=elements-platform.rwelements](https://marketplace.visualstudio.com/items?itemName=elements-platform.rwelements)
2. Click **Install** or open it directly in Visual Studio Code.
3. Restart VSCode (if needed) to enable full functionality.

Once installed, the extension activates automatically when you open an Elements project.
