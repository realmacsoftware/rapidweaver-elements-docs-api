# TextStyles

Add Weight to your element.

## Usage

```json
{
  "globalControl": "TextStyles"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `textStylesFontWeight` |
| format | string | `font-[{{value}}]` |
| UI Type | string | `slider` |
| default | string | `400` |
