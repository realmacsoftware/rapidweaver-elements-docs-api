# TextColor

Add Color to your element.

## Usage

```json
{
  "globalControl": "TextColor"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalTextColor` |
| format | string | `text-{{value}}` |
| UI Type | string | `slider` |
