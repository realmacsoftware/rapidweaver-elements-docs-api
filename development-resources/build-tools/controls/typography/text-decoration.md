# TextDecoration

Add Underline to your element.

## Usage

```json
{
  "globalControl": "TextDecoration"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `textDecoration` |
| format | string | `decoration-[{{value}}px]` |
| UI Type | string | `select` |
| default | string | `no-underline` |
