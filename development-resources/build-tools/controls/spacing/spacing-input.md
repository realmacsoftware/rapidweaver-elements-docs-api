# SpacingInput

Add Margin & Padding to your element.

## Usage

```json
{
  "globalControl": "SpacingInput"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalSpacingEnabled` |
| UI Type | string | `switch` |

## Includes

This control includes the following nested controls:

- `Margin`
- `Padding`
