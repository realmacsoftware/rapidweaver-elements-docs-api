# AlignItems

Add Items to your element.

## Usage

```json
{
  "globalControl": "AlignItems"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `alignItems` |
| format | string | `items-{{value}}` |
| UI Type | string | `select` |
| default | string | `[normal]` |
