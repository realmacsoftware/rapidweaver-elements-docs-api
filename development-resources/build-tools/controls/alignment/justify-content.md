# JustifyContent

Add Content to your element.

## Usage

```json
{
  "globalControl": "JustifyContent"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `justifyContent` |
| format | string | `justify-{{value}}` |
| UI Type | string | `select` |
| default | string | `normal` |
