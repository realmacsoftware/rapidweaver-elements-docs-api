# JustifyItems

Add Items to your element.

## Usage

```json
{
  "globalControl": "JustifyItems"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `justifyItems` |
| format | string | `justify-items-{{value}}` |
| UI Type | string | `select` |
| default | string | `normal` |
