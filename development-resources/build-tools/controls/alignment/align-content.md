# AlignContent

Add Content to your element.

## Usage

```json
{
  "globalControl": "AlignContent"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `alignContent` |
| format | string | `content-{{value}}` |
| UI Type | string | `select` |
| default | string | `normal` |
