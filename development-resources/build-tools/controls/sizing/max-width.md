# MaxWidth

Add Max Width to your element.

## Usage

```json
{
  "globalControl": "MaxWidth"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalMaxWidth` |
| format | string | `max-w-{{value}}` |
| UI Type | string | `themeSpacing` |
