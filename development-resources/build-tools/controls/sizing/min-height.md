# MinHeight

Add Min Height to your element.

## Usage

```json
{
  "globalControl": "MinHeight"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalMinHeight` |
| format | string | `min-h-{{value}}` |
| UI Type | string | `themeSpacing` |
