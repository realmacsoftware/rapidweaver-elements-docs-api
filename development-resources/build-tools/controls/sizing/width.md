# Width

Add Width to your element.

## Usage

```json
{
  "globalControl": "Width"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalWidth` |
| format | string | `w-{{value}}` |
| UI Type | string | `themeSpacing` |
