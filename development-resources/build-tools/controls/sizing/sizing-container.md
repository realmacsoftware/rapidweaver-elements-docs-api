# SizingContainer

Add Size to your element.

## Usage

```json
{
  "globalControl": "SizingContainer"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalWidthType` |
| UI Type | string | `select` |
| default | string | `full` |

## Includes

This control includes the following nested controls:

- `Width`
- `Height`
- `MinWidth`
- `MaxWidth`
- `MinHeight`
- `MaxHeight`
