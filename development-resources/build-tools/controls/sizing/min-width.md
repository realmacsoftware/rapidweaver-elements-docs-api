# MinWidth

Add Min Width to your element.

## Usage

```json
{
  "globalControl": "MinWidth"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalMinWidth` |
| format | string | `min-w-{{value}}` |
| UI Type | string | `themeSpacing` |
