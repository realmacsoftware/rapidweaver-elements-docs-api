# Height

Add Height to your element.

## Usage

```json
{
  "globalControl": "Height"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalHeight` |
| format | string | `h-{{value}}` |
| UI Type | string | `themeSpacing` |
