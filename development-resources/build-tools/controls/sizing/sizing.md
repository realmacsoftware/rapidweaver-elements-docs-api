# Sizing

Add Size to your element.

## Usage

```json
{
  "globalControl": "Sizing"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalSizingMinMaxEnabled` |
| UI Type | string | `switch` |

## Includes

This control includes the following nested controls:

- `Width`
- `Height`
- `MinWidth`
- `MaxWidth`
- `MinHeight`
- `MaxHeight`
