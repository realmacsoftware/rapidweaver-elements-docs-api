# MaxHeight

Add Max Height to your element.

## Usage

```json
{
  "globalControl": "MaxHeight"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalMaxHeight` |
| format | string | `max-h-{{value}}` |
| UI Type | string | `themeSpacing` |
