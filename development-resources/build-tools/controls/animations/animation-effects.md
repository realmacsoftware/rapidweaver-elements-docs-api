# AnimationEffects

Add Preview to your element.

## Usage

```json
{
  "globalControl": "AnimationEffects"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalScrollAnimationPreviewInEditor` |
| UI Type | string | `segmented` |
| default | string | `start` |
