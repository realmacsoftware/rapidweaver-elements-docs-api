# Delay

Add Delay to your element.

## Usage

```json
{
  "globalControl": "Delay"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalTransitionsDelay` |
| format | string | `delay-[{{value}}ms]` |
| UI Type | string | `number` |
