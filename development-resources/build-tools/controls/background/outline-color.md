# OutlineColor

Add Color to your element.

## Usage

```json
{
  "globalControl": "OutlineColor"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalOutlineColor` |
| format | string | `outline-{{value}}` |
| UI Type | string | `slider` |
