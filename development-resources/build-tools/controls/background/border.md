# Border

Add Style to your element.

## Usage

```json
{
  "globalControl": "Border"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `borderStyle` |
| format | string | `border-{{value}}` |
| UI Type | string | `segmented` |
| default | string | `none` |
