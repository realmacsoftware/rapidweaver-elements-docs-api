# Background_Image

Add Image to your element.

## Usage

```json
{
  "globalControl": "Background_Image"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalBgImageResource` |
| format | string | `bg-{{value}}` |
| UI Type | string | `select` |
| default | string | `center` |
