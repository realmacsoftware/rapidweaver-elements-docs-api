# Background_SVG

Add SVG to your element.

## Usage

```json
{
  "globalControl": "Background_SVG"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalBgSVGResource` |
| format | string | `text-{{value}} [&>svg]:fill-current` |
| UI Type | string | `slider` |
