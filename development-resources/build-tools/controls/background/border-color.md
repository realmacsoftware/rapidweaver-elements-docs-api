# BorderColor

Add Color to your element.

## Usage

```json
{
  "globalControl": "BorderColor"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalBordersColor` |
| format | string | `border-{{value}}` |
| UI Type | string | `slider` |
