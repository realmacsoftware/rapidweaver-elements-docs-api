# BackgroundImage

Add Image to your element.

## Usage

```json
{
  "globalControl": "BackgroundImage"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `bgImage` |
| format | string | `bg-{{value}}` |
| UI Type | string | `select` |
| default | string | `center` |
