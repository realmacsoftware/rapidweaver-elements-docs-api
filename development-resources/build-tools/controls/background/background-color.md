# BackgroundColor

Add Color to your element.

## Usage

```json
{
  "globalControl": "BackgroundColor"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `bgColor` |
| format | string | `bg-{{value}}` |
| UI Type | string | `slider` |
