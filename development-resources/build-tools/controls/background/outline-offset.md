# OutlineOffset

Add Offset to your element.

## Usage

```json
{
  "globalControl": "OutlineOffset"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalOutlineOffset` |
| format | string | `outline-offset-[{{value}}px]` |
| UI Type | string | `number` |
