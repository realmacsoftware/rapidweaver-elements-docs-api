# BorderStyle

Add Style to your element.

## Usage

```json
{
  "globalControl": "BorderStyle"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalBordersStyle` |
| format | string | `border-{{value}}` |
| UI Type | string | `segmented` |
| default | string | `solid` |
