# OutlineWidth

Add Width to your element.

## Usage

```json
{
  "globalControl": "OutlineWidth"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalOutlineWidth` |
| format | string | `outline-[{{value}}px]` |
| UI Type | string | `number` |
