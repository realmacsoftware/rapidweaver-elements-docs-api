# BorderRadius

Add Radius to your element.

## Usage

```json
{
  "globalControl": "BorderRadius"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalBordersRadius` |
| format | string | `{{value}}` |
| UI Type | string | `themeBorderRadius` |
