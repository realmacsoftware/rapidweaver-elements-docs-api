# Overlay_Image

Add Image to your element.

## Usage

```json
{
  "globalControl": "Overlay_Image"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalOverlayImageResource` |
| format | string | `bg-{{value}}` |
| UI Type | string | `select` |
| default | string | `center` |
