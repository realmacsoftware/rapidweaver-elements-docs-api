# Overlay_Color

Add Color to your element.

## Usage

```json
{
  "globalControl": "Overlay_Color"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalOverlayColor` |
| format | string | `bg-{{value}}` |
| UI Type | string | `slider` |
