# Layout

Add General to your element.

## Usage

```json
{
  "globalControl": "Layout"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| UI Type | string | `heading` |

This control includes the following nested controls:

- `Position`
- `ZIndex`
- `TopRightBottomLeft`
- `Hidden`
- `Visibility`
- `Overflow`
- `Isolation`
