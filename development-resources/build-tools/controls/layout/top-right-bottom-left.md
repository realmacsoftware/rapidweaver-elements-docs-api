# TopRightBottomLeft

Add Top / Right / Bottom / Left to your element.

## Usage

```json
{
  "globalControl": "TopRightBottomLeft"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalLayoutTopRightBottomLeftType` |
| format | string | `top-{{value}}` |
| UI Type | string | `segmented` |
| default | string | `none` |
