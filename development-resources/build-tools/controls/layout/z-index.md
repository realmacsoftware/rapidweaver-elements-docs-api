# ZIndex

Add Z-Index to your element.

## Usage

```json
{
  "globalControl": "ZIndex"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalLayoutZIndexType` |
| format | string | `z-[{{value}}]` |
| UI Type | string | `segmented` |
