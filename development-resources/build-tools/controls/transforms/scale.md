# Scale

Add Scale to your element.

## Usage

```json
{
  "globalControl": "Scale"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalTransformScale` |
| format | string | `scale-[{{value}}%]` |
| UI Type | string | `number` |
