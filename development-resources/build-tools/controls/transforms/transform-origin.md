# TransformOrigin

Add Origin to your element.

## Usage

```json
{
  "globalControl": "TransformOrigin"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalTransformOrigin` |
| format | string | `origin-{{value}}` |
