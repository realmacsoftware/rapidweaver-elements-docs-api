# Columns

Add Columns to your element.

## Usage

```json
{
  "globalControl": "Columns"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalGridItemColSpan` |
| format | string | `col-span-{{value}}` |
| UI Type | string | `slider` |

## Includes

This control includes the following nested controls:

- `ColStart`
- `ColEnd`
