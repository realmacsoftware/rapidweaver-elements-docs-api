# FlexDirection

Add Direction to your element.

## Usage

```json
{
  "globalControl": "FlexDirection"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `flexDirection` |
| UI Type | string | `select` |
| default | string | `flex-col` |
