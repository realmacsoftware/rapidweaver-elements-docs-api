# FlexItem

Add Alignment to your element.

## Usage

```json
{
  "globalControl": "FlexItem"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalFlexItemFlex` |
| format | string | `flex-{{value}}` |
| UI Type | string | `segmented` |
| default | string | `none` |

## Includes

This control includes the following nested controls:

- `AlignSelf`
- `JustifySelf`
- `Order`
