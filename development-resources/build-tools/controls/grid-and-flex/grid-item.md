# GridItem

Add Columns to your element.

## Usage

```json
{
  "globalControl": "GridItem"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| UI Type | string | `heading` |

## Includes

This control includes the following nested controls:

- `Columns`
- `Rows`
- `AlignSelf`
- `JustifySelf`
