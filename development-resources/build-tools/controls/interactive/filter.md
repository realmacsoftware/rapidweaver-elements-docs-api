# Filter

Add Filtering to your element.

## Usage

```json
{
  "globalControl": "Filter"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalFilterEnable` |
| UI Type | string | `select` |
| default | string | `parent` |
