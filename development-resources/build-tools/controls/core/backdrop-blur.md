# BackdropBlur

Add Blur to your element.

## Usage

```json
{
  "globalControl": "BackdropBlur"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalFiltersBackdropBlur` |
| format | string | `backdrop-blur-[{{value}}px]` |
| UI Type | string | `number` |
