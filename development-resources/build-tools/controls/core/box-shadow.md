# BoxShadow

Add Box Shadow to your element.

## Usage

```json
{
  "globalControl": "BoxShadow"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalBoxShadow` |
| UI Type | string | `themeShadow` |
