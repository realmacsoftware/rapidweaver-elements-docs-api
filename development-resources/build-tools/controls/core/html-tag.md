# HTMLTag

Add HTML Tag to your element.

## Usage

```json
{
  "globalControl": "HTMLTag"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalHTMLTag` |
| UI Type | string | `select` |
| default | string | `default` |
