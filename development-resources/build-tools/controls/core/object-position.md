# ObjectPosition

Add Position to your element.

## Usage

```json
{
  "globalControl": "ObjectPosition"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `objectPosition` |
| UI Type | string | `select` |
| default | string | `object-center` |
