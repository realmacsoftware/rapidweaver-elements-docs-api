# MenuItem

Add Font to your element.

## Usage

```json
{
  "globalControl": "MenuItem"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalMenuItemFontFamily` |
| format | string | `font-[{{value}}]` |
| UI Type | string | `segmented` |
| default | string | `400` |
