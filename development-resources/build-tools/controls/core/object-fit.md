# ObjectFit

Add Object Fit to your element.

## Usage

```json
{
  "globalControl": "ObjectFit"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `objectFit` |
| format | string | `object-{{value}}` |
| UI Type | string | `select` |
| default | string | `cover` |
