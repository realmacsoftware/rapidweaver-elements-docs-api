# DropShadow

Add Drop Shadow to your element.

## Usage

```json
{
  "globalControl": "DropShadow"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalFiltersDropShadow` |
| UI Type | string | `themeShadow` |
