# Blur

Add Blur to your element.

## Usage

```json
{
  "globalControl": "Blur"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalFiltersBlur` |
| format | string | `blur-[{{value}}px]` |
| UI Type | string | `number` |
