# Opacity

Add Opacity to your element.

## Usage

```json
{
  "globalControl": "Opacity"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalOpacity` |
| format | string | `opacity-[{{value}}%]` |
| UI Type | string | `slider` |
