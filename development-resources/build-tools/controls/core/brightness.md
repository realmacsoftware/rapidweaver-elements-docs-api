# Brightness

Add Brightness to your element.

## Usage

```json
{
  "globalControl": "Brightness"
}
```

## Properties

| Property | Type | Value |
|----------|------|-------|
| id | string | `globalFiltersBrightness` |
| format | string | `brightness-[{{value}}%]` |
| UI Type | string | `number` |
