# RevealAnimations

Pre-defined values for RevealAnimations.

## Usage

```json
{
  "select": {
    "use": "RevealAnimations"
  }
}
```

## Default

`fade`

## Values

| Value | Title |
|-------|-------|
| `fade` | Fade |
| `slide` | Slide |
| `zoom` | Zoom |
| `lightSpeed` | Light Speed |
| `rotate` | Rotate |
