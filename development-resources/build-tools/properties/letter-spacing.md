# LetterSpacing

Pre-defined values for LetterSpacing.

## Usage

```json
{
  "select": {
    "use": "LetterSpacing"
  }
}
```

## Default

`normal`

## Values

| Value | Title |
|-------|-------|
| `tighter` | Tighter |
| `tight` | Tight |
| `normal` | Normal |
| `wide` | Wide |
| `wider` | Wider |
| `widest` | Widest |
