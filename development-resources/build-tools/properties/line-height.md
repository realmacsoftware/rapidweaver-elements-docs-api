# LineHeight

Pre-defined values for LineHeight.

## Usage

```json
{
  "select": {
    "use": "LineHeight"
  }
}
```

## Default

`normal`

## Values

| Value | Title |
|-------|-------|
| `none` | None |
| `tight` | Tight |
| `snug` | Snug |
| `normal` | Normal |
| `relaxed` | Relaxed |
| `loose` | Loose |
