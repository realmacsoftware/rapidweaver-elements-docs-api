# FontWeight

Pre-defined values for FontWeight.

## Usage

```json
{
  "select": {
    "use": "FontWeight"
  }
}
```

## Default

`400`

## Values

| Value | Title |
|-------|-------|
| `100` | Thin |
| `200` | Extra Light |
| `300` | Light |
| `400` | Normal |
| `500` | Medium |
| `600` | Semi Bold |
| `700` | Bold |
| `800` | Extra Bold |
| `900` | Black |
