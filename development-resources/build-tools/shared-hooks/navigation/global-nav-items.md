# globalNavItems

Utility function for navigation.

## Signature

```javascript
globalNavItems(rw, isActive = false)
```

## Parameters

| Parameter | Description |
|-----------|-------------|
| `rw` | - |
| `isActive` | - |

## Usage

```javascript
function transformHook(rw) {
    const result = globalNavItems(rw);
    return { result };
}
```
