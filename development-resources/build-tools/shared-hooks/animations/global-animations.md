# globalAnimations

Utility function for animations.

## Signature

```javascript
globalAnimations(app)
```

## Parameters

| Parameter | Description |
|-----------|-------------|
| `app` | - |

## Usage

```javascript
const transformHook = (rw) => {
    const animation = globalAnimations(rw);
    // animation.isEnabled, animation.data, etc.
};
```
