# Image

Displays an image dropwell in the inspector.

{% tabs %}
{% tab title="Control Example" %}
```json
{
  "title": "Hero Image",
  "id": "heroImage",
  "image": {}
}
```
{% endtab %}

{% tab title="Group Example" %}
```json
{
  "groups": [{
    "title": "Hero Media",
    "properties": [{
      "title": "Hero Image",
      "id": "heroImage",
      "image": {}
    }]
  }]
}
```
{% endtab %}
{% endtabs %}
